Data Migration Guide

This system replaces scattered data sources:

  • WhatsApp Groups: Export chat data and manually create CSV with alumni information
  • Excel Sheets: Save as CSV and import directly
  • Email Lists: Export contacts and format as CSV
  • Google Forms: Export responses as CSV and import
This centralized system ensures no alumni data is lost and provides easy access for long-term engagement.
